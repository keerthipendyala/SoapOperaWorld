Academic Projects

Soap Opera World ,Web Development, Northeastern University

Developed an online TV show DVD store ,Soap Opera World that allows users to obtain information about various soap operas, review, comment and buy them. It was developed using MongoDB, AngularJS and node.js. Implemented RESTful APIs using Express.js. Provided secured authentication using Passport.js. APIs used: TMDB for TV Show Data

Web Based Search Engine, Information Retrieval,Northeastern University

Crawled 1000 unique webpages, given seed page, using BFS traversal while maintaining a politeness policy of 1 second between fetching each page using Java thread. Parsed HTML pages using Jsoup library to fetch links from anchor tags. Removed punctuation marks and stop words using Java regex to perform topical analysis of each webpage. Built a simple indexer using Vector Space Similarity algorithm to return links to top 100 webpages given a search query string as input to compare the terms in a query and the webpage itself.
