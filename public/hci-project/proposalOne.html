Proposal 1 - NUTransit


Project Idea

This web application is a dashboard with live updates on nearest transport options available and whether you can reach them on time. For NU students, the options for public transport are Orange Line, Green Line , Bus and RedEye. The website at any point shows transport in these categories and their times of arrival along with an indication of whether the student will be able to make it in time to catch it based on his/her current location. The student can process all this information at a glance and head towards the nearest possible option without wasting any time.

Technologies and Comparison with existing systems

Technology
I would like to implement this as a MEAN stack web application using Google Distance Matrix and MBTA APIs
Existing Systems
1) Google Maps shows the fastest route and the transport options available. However there is no comparison between these options at a quick glance. The user has to mentally calculate if he will make it on time to catch the train and switch between tabs while doing so.
Key Benefits
1) Designed to help NU students evaluate the quickest possible mode of transport in the vicinity of the campus.
2) A indication of whether they will reach on time to catch the train/bus is useful in making this choice.
